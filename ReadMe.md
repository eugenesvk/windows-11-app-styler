## Install
  - Install [Windhawk](https://windhawk.net)
  - Settings App:
    - (temporary inconvenience) create a new mod in the Windhawk app, paste the [latest source code version](https://github.com/eugenesvk/windhawk-mods/blob/e/mods/windows-11-app-styler-settings.wh.cpp), compile, and enable
    - (when it's published) [search here](https://windhawk.net/mods) or in your **Explore** tab in the Windhawk app and install there
